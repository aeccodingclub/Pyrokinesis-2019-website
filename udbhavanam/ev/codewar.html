Introduction

Code War is where developers achieve code mastery through challenges. It is where problem statements are given to the participants and the participants are to find the solutions of the same in different programming languages provided, within a finite period of time. The problem statements are strictly based on real life situations. Shortlisted candidates would be awarded and certificates will be given to the participants.
